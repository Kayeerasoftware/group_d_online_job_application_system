## 🔄 User Journey

```
START
  ↓
Navigate to /seeker/profile/edit
  ↓
Page loads with current profile data
  ↓
See completion percentage and guidance
  ↓
Fill in/update form fields
  ↓
Real-time feedback updates
  ├─ Skills preview updates
  ├─ Progress bar updates
  └─ File name updates
  ↓
Upload resume (optional)
  ↓
Review profile snapshot
  ↓
Click "Save Changes"
  ↓
Form validates
  ├─ Check required fields
  ├─ Validate file type
  └─ Validate file size
  ↓
Data saves to database
  ↓
Success message displays
  ↓
Redirect to profile edit page
  ↓
END
```
